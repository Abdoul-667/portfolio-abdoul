# Portfolio Abdoul Wahab Maïga

## Description
Portfolio développé avec Next.js avec authentification, projets et témoignages.

## Technologies
- Next.js
- React
- Redux Toolkit
- Tailwind CSS

## Fonctionnalités
- Login / Register
- Protection des pages
- Affichage des projets
- Témoignages (ajout)

## Lancer
npm install
npm run dev
